# 🖼️ Image Steganography Using C

## 📌 Project Overview

Developed an Image Steganography application in C that enables secure hiding and retrieval of secret messages within image files. The project implements steganography techniques to conceal confidential text data inside a BMP image without visibly altering the image, ensuring secure communication and data protection.

## 🚀 Features

* 🔒 Hide secret text messages inside BMP images
* 🔍 Extract hidden messages from encoded images
* 🖼️ Supports BMP image format
* 📂 Encode and decode operations through command-line interface
* 🛡️ Maintains image quality after data embedding
* ✅ Validation of input image and secret file formats
* ⚡ Efficient bit-level data manipulation using LSB (Least Significant Bit) technique

## 🛠️ Technologies Used

* Programming Language: C
* Concepts: File Handling, Bit Manipulation, Pointers, Structures
* Image Processing: BMP File Format Handling
* Security: Data Hiding using LSB Steganography
